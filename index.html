Sobre Mim

Segurança com propósito.
Automação com impacto.

Sou finalista de Engenharia Informática com uma paixão profunda pela interseção entre redes, segurança e automação. Acredito que a melhor defesa é uma infraestrutura que se protege e repara a si própria.

O meu percurso combina experiência prática com SIEMs (Wazuh, Splunk), firewalls (Palo Alto, Fortinet), e ferramentas de Infrastructure as Code (Terraform, Bicep, Ansible) para construir sistemas que não só detetam ameaças — mas respondem autonomamente.

Procuro funções como Security Automation Engineer, Cloud Security Engineer ou SOC Analyst L1/L2 onde possa aplicar estas competências em ambientes reais de produção.
